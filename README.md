# Modern Interactive Portfolio

Este es mi portafolio profesional personal, diseñado con una estética moderna de **Glassmorphism** y una temática espacial.

## Características

- **Terminal Interactiva**: Una shell funcional (vía JS) que permite explorar mi perfil mediante comandos.
- **Efectos Visuales Avanzados**:
  - Cursor personalizado con seguidor.
  - Fondo de estrellas animado.
  - "Ojos" interactivos que siguen el movimiento del mouse.
- **Sección de Proyectos**: Galería con modales animados y carrusel de imágenes para mostrar mis desarrollos en Django, C++, y más.
- **Diseño Responsivo**: Optimizado para dispositivos móviles y escritorio.

## Tecnologías Utilizadas

- **HTML5** (Semántico)
- **CSS3** (Variables, Animaciones, Backdrop-filter)
- **JavaScript** (ES6+, MutationObserver para la terminal)
- **FontAwesome** para la iconografía.

## Instalación

Simplemente clona el repositorio y abre `index.html` en tu navegador.
```bash
git clone https://github.com/Tokyossz/Modern-Portfolio.git
```
